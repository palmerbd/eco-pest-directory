# Link Building Target List — Ballroom Dance Directory

Generated: 2026-04-20
Context: April GSC review flagged link-building as High-Impact/High-Effort. Site's average SERP position is 15.3; a handful of relevant DA 30+ backlinks should be enough to measurably lift rankings for mid-volume queries like "ballroom dance classes near me" and city-level studio terms.

## Outreach strategy (brief)

Send 5–8 pitches per week. Track in a simple spreadsheet: target, URL, contact, date sent, follow-up date, status, link placed? For each target, personalize the opening line — reference a recent post, a specific studio they've covered, or a local event. The pitch itself should offer something: a curated studio list for their city, a data point from the directory ("we track 900+ private studios nationally"), a guest post, or a reciprocal mention.

Prioritize DA 30+ sites first, but don't skip small local blogs — a relevant local link from a DA 20 wedding planner in Boulder is worth more for city rankings than a high-DA generic mention.

## Outreach cadence

- **Week 1–2**: Tier 2 wedding blogs (highest conversion value). Send 5 custom pitches.
- **Week 3–4**: Tier 1 industry associations. These take longer to reply but carry the most authority.
- **Week 5–6**: Tier 3 local lifestyle (start with Boulder, Bellevue, Fox Valley — cities where BDD already has ranking signal).
- **Week 7–8**: Tier 4 low-effort submissions. Batch these — 10–15 per session.
- **Ongoing**: Tier 5 podcast/YouTube pitches; one per week.
